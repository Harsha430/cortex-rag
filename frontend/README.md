# CortexRAG Frontend

The premium "Neural" interface for CortexRAG. Built with React and Vite for maximum responsiveness and high-depth aesthetics.

## 🎨 Design System
- **Theme**: High-depth Neural Dark Mode.
- **Components**: Floating Console, Acrylic Sidebar, Glassmorphic Bubbles.
- **Animations**: Entrance transitions and neural pulse loading states.

## 🚀 Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## 📦 Build
To create a production build:
```bash
npm run build
```
